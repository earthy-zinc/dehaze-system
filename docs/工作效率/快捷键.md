## 浏览器快捷键

| 快捷键名称                   | 说明                           |
| ---------------------------- | ------------------------------ |
| CTRL+N                       | 打开新的标签页                 |
| CTRL+W/CTRL+F4               | 关闭标签页                     |
| CTRL+TAB/CTRL+PAGEUP         | 跳转到下一个标签页             |
| CTRL+SHIFT+TAB/CTRL+PAGEDOWN | 跳转到上一个标签页             |
| CTRL+1-8,9                   | 跳转到指定的页面，最后一个页面 |
| F5                           | 刷新网页                       |
