# WebPack面试题

## WebPack、grunt、gulp

Grunt、Gulp是基于任务运行的工具，他们会自动执行指定的任务，就像流水线，把资源放上去然后通过不同插件进行加工。

Webpack基于模块化打包的工具：自动化处理模块，当webpack处理应用程序时，它会递归地构建一个依赖关系图，其中包含应用程序需要的每一模块，然后将这些模块打包成一个过着多个bundle

## Webpack构建流程

webpack的运行是一个串行的过程。

1. 初始化参数
2. 开始编译
3. 确定入口
4. 编译模块
5. 完成模块编译
6. 输出资源
7. 输出完成
